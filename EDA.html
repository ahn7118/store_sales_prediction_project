Data 설명¶

  • store_id : 각 파일에서의 상점 고유 번호 (해당기한 내 개업 혹은 폐업한 상점도 있음)
  • date : 거래일자
  • time : 거래시간
  • card_id : 카드 번호의 hash 값
  • amount : 매출 금액, 음수인 경우 취소 거래
  • installments : 할부 개월 수. 일시불은 0
  • days_of_week : 요일, 월요일이 0, 일요일은 6
  • holiday : 1이면 공휴일, 0이면 공휴일 아님
